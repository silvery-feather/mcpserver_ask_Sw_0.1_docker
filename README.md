## macOS / Linux

```bash
git clone https://github.com/silvery-feather/mcpserver_ask_Sw_0.1_docker  
cd mcpserver_ask_Sw_0.1_docker  

export DASHSCOPE_API_KEY="你的Key"  
docker compose up -d --build  
```

## Windows PowerShell

```bash
git clone https://github.com/silvery-feather/mcpserver_ask_Sw_0.1_docker  
cd mcpserver_ask_Sw_0.1_docker  

$env:DASHSCOPE_API_KEY="替换为你的Key"  
docker compose up -d --build  

```
